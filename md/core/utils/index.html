Table

Bases: Generic[ROWKEYT, COLKETT, ITEM]

使用示例

table = Table() table.add_column("Name") table.add_column("Age") table.add_row("Row1") table.add_row("Row2")

增加数据

table["Row1", "Name"] = "Alice" table["Row1", "Age"] = 25 table["Row2", "Name"] = "Bob" table["Row2", "Age"] = 30

print(table.get_row("Row1")) # 输出: {'Name': ['Alice'], 'Age': [25]} print(table.get_column("Age")) # 输出: {'Row1': [25], 'Row2': [30]}

删除数据

del table["Row1", "Age"] print(table.get_row("Row1")) # 输出: {'Name': ['Alice'], 'Age': []}
